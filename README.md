<h1>Testes de Funções</h1>
<br>
<p>Aplicação desenvolvida com HTML, CSS e JavaScript.</p>
<br>
<p>Interface simples, focado nas funcionalidades. Ela contém:</p>
<ul>
  <li>3 botões iniciais onde o usuário seleciona qual função quer executar;</li>
  <li>O primeiro botão, ao ser clicado, abre um input que faz o cálculo fatorial do número inserido;</li>
  <li>O segundo, verifica se o ano inserido é ou não um ano bissexto;</li>
  <li>O terceiro, verifica se a palavra inserida é um palíndromo.</li>
</ul>
